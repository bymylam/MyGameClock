# MyGameClock ⏱️

MyGameClock is an iOS app built with Swift and UIKit that helps two players track game time with customizable timers. It features start/stop controls, turn indicators, reset functionality, and a time editor screen, providing a user-friendly experience while demonstrating UI design, timer logic, and state management skills.

## Features
- Start/Stop game clock
- Turn indicators for top and bottom players
- Reset timers to initial value
- Customizable starting time via Time Editor screen
- Real-time countdown for each player
- Visual feedback when a player runs out of time

## Tech Stack
- Swift
- UIKit (UIButton, UILabel, UIPickerView, UIViewController)
- Xcode
- MVC Design Pattern
- Timer & State Management

## How to Run
1. Clone this repository: https://github.com/bymylam/MyGameClock
2. Open `MyGameClock.xcodeproj` in Xcode
3. Build and run the app on a simulator or device
